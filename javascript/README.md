# RoutePlex JavaScript Examples

Working JavaScript examples for the [RoutePlex](https://routeplex.com) API.

## Setup

```bash
npm install
export ROUTEPLEX_API_KEY="your-api-key-here"
```

## Examples

| File | Description |
|------|-------------|
| `speed_strategy.js` | Fastest response with native fetch |
| `quality_strategy.js` | Best output for complex tasks |
| `cost_strategy.js` | Minimize expenses for high-volume |
| `web_search.js` | Real-time data via auto web search |
| `openai_compatible.js` | OpenAI SDK drop-in replacement |

## Run

```bash
node speed_strategy.js
node openai_compatible.js
```
